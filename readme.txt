◆何ができるの？
・座る。
タスクバー／アクティブウインドウに座ります。
「どこに座っても良いよ」選択で浮きます。

・１時間半までの遠征をタイマ管理。
ゲーム中で遠征を出したと同時にメニューから遠征タイマをクリックして下さい。
遠征帰還時にお知らせしてくれます。（ゴーストが終了した時点でリセットされます）

・各資材別オススメ遠征リスト
メニューの遠征指南からどうぞ。
（リストはＷｉｋｉを参考に独断と偏見で選んであります。）
